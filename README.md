## Used technologies

- Expo (React Native)

## Setup

`npm install`

## Development

`npm run start:android`
`npm run start:ios`

## Build App

This builds the full standalone app,

`npm run build:android`
`npm run build:ios`

## Updates

### Expo SDk and dependencies

`npm run upgrade`

### Publish OTA Update of the App

`npm run publish`

The App on the Phone checks then automaticly for updates
